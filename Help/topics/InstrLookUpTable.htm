Tabela de Busca

Inserir tabela de busca na posição do cursor.

Esta instrução define uma tabela de valores (até o limite de 32, conforme definido no campo “Contador”) que são atribuídos à variável fornecida no campo “Destino”. A escolha de qual valor será atribuído é feita pela variável no campo “Índice”. Esta variável define a posição na tabela que deve ser atribuído. Se o índice for inválido, a variável no campo “Destino” permanece inalterada.
