Ler String Formatada

Ler String Formatada: Inserir ler string formatada na posição do cursor.




Essa instrução lê a interface serial RS-485, checando o padrão definido no campo “String” e, caso o valor lido da serial coincida com o padrão, o valor interpretado no padrão é atribuído à variável no campo “Destino.
