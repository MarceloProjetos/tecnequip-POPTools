Teclas de Atalho do POPTools

Atalhos:

POPtools possui diversas teclas de atalho, mas lembre-se que a barra de ferramentas  do POPTools possui todos os componentes para a edição de seu programa. Segue abaixo a lista de Teclas de Atalho aceitas pelo programa:



Teclas
 O que faz?
F1 Manual do POPTools
F2 Configurações do projeto
F3 Exibir caixa de diálogo de busca
Shift+F3 Busca a próxima ocorrência
F4 Fecha a janela ou Arquivo selecionado
F5 Compilar Diagrama Ladder
F6   Compilar Diagrama Ladder e gravar no CLP
F7 Entrar no Modo de Simulação
F8 Executa a janela de depuração
Enter Modo Normal: Exibe propriedades do objeto selecionado
Modo Simulação: Executa um ciclo
Shift+seta⇧ Mover a linha atual para cima, somente se houver uma linha acima
Shift+seta⇩ Mover a linha atual para baixo, somente se houver uma linha abaixo
 ←Backspace Exclui o item selecionado. Se a linha estiver vazia exclui a linha.
Delete Exclui o item selecionado. Se a linha estiver vazia exclui a linha.
Home Seleciona o primeiro objeto da linha atual
End Seleciona o último objeto da linha atual
ESC Sair do modo de simulação
A Inserir Atraso para Ligar
Shift+A Inserir Atraso para Desligar
Ctrl+A Inserir leitura do A/D
B Inserir Bobina
Shift+B Inserir Checar bit
Ctrl+B Inserir Liga/Desliga bit
C Inserir um contato
Shift+C Inserir contador circular
Ctrl+C Copiar um item selecionado para memória
D Inserir contador Decremental
Ctrl+D Inserir uma escrita no D/A
E Inserir Leitura de Encoder de Quadratura
Shift+E Inserir Reset de Encoder de Quadratura
Ctrl+E Abrir a janela de diálogo "Exportar como Texto"
I Inserir Contador Incremental
  ESPAÇO 
Seleciona modo Normal para Bobinas e Contatos
N Seleciona modo Invertido para Bobinas e Contatos
Ctrl+N Abrir um novo Diagrama Ladder
Ctrl+O Abrir um Diagrama Ladder existente
P Modo Normal: Inserir Saída PWM
Modo Simulação: Pausar
R Seleciona modo Desligar para Bobinas
R   Reiniciar a simulação em tempo real
Shift+R Adicionar Rampa de Aceleração/Desaceleração
Ctrl+R Iniciar simulação em tempo real
S Seleciona modo Ligar para Bobinas
Ctrl+S Salvar o Diagrama Ladder
T Inserir Atraso Retentivo para Ligar
U Inserir Escrita USS
Shift+U Inserir Leitura USS
Ctrl+V Colar na posição do cursor o item previamente copiado para a memória
X
 Inserir Escrever Registrador ModBUS Ethernet
Shift+X Inserir Ler Registrador ModBUS Ethernet
Ctrl+X Cortar o objeto selecionado
Y Inserir Escrita no NS-600 Yaskawa
Shift+Y Inserir Leitura de NS-600 Yaskawa
Ctrl+Y Refazer a última alteração desfeita
Z Inserir Zerar (para Temporizadores e Contadores)
Shift+Z Inserir Zerar Encoder Incremental
Ctrl+Z Desfazer a ultima alteração
4
 Inserir Escrever Registrador ModBUS RS-485
Shift+4 Inserir Ler Registrador ModBUS RS-485
INSERT
 Insere uma nova linha abaixo da linha atual
Shift+INSERT Insere uma nova linha acima da linha atual
 ; ponto e virgula
 Inserir comentário
+
 Inserir Somar
-
 Inserir Subtrair
* Inserir Multiplicar
\ Inserir Borda de Descida
/
 Inserir Borda de Subida
Shift+/ Inserir Dividir
=
 Inserir Condicional se Igual
> Inserir Condicional Maior
Shift+> Inserir Condicional Maior ou Igual
< Inserir Condicional Menor
Shift+< Inserir Condicional Menor ou Igual
! Inserir Condicional Diferente
