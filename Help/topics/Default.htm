Introdução

POPtools é uma aplicação poderosa que tem como objetivo criar automações profissionais com uma interface simples e moderna.  Esse documento tem como adjetivo facilitar o entendimento das funções e a utilização adequada do CLP POP-7.POPTools é uma ferramenta necessária para gerar o programa, que vai coordenar e seguenciar as operações que o CLP deve executar.

CLP (Controlador Lógico Programável) Sua definição segundo a ABNT "É um equipamento eletrônico digital com hardware e software compatíveis com aplicações industriais". Ele utiliza uma memória programável para armazenamento instruções para aplicações especificas, tais como lógica, seqüenciamento, temporizações, contagem e aritmética, para controlar através de módulos de entrada e saídas vários tipos de maquinas ou processos industriais.

Caracteristicas do CLP:

  • Dispositivo de controle de fácil e rápida programação
  • Capacidade de operação em ambiente industrial;
  • Compatibilidade com diferentes tipos de sinais de entrada e saída;

  • Possibilidade de monitoramento do processo, através da comunicação com computadores;

Caracteristicas do POPTools:

  • Baseado em logica de relé, com contatos e bobinas, por ser a linguagem de programação de CLP mais simples de ser assimilada por quem já tenha conheciamento de circuitos de comando eletrico;
  • É feito de varios circuitos dispostos de maneira horizontal, com bobinas na extremidade direita, alimentados por duas barras verticais laterais. Por esse formato e que recebe o nome de ladder que significa escada, em inglês.
  • Cada uma das linhas horizontais é uma setença lógica onde os contatos são as entradas das sentenças, as bobinas são as saidas e a associação dos contatos é a lógica.
